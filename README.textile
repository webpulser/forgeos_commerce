h1. ForgeosCommerce

ForgeosCommerce is an extension of Ruby on Rails for e-commerce development

VERSION : 0.9.6

h1. Prerequisites

h2. Plugins

* ForgeosCore
* ForgeosCMS

h2. Initialize

  <pre><code>
  $ ./script/generate plugin_migration forgeos_commerce
  </code></pre>

Launch FogeosCommerce Install :

  <pre><code>
  $ rake forgeos:commerce:install
  </code></pre>
