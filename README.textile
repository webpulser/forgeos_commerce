h1. ForgeosCommerce

ForgeosCommerce is an extension of Ruby on Rails for e-commerce development 

VERSION : 0.9.0-rc2

h1. Prerequisites

h2. Plugins

* ForgeosCore
* ForgeosCMS

h2. Initialize

  <pre><code>
  $ ./script/generate migration install_forgeos_commerce_plugin
  </code></pre>

Edit the generated file like this :

  <pre><code>
  class InstallForgeosCommerce < ActiveRecord::Migration
    def self.up
      migrate_plugin :forgeos_commerce, LAST_MIGRATION
    end

    def self.down
      migrate_plugin :forgeos_commerce, 0
    end
  end
  </code></pre>

Note: LAST_MIGRATION is the number of the last database migration of ForgeosCommerce.

Launch FogeosCommerce Install :

  <pre><code>
  $ rake forgeos:commerce:install
  </code></pre>
